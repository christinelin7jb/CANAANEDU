⚙ One-time Firebase Setup (5 min)
1
Go to console.firebase.google.com → sign in with your company Google account → click Add Project
2
In the project: go to Build → Firestore Database → Create in production mode. Then Authentication → Sign-in method → Google → Enable
3
Go to Project Settings → Your Apps → Web App (</>) → Register app → copy the firebaseConfig object
4
Paste each value below and click Save. This is stored locally on each consultant's browser — share these values with your team.
